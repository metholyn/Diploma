# Розробка Веб-застосунку для бібліотеки

**Дипломник:** Переверзов Марк Андрійович (6.1212-пзс)  
**Тема:** Розробка Веб-застосунку для бібліотеки з інтеграцією Spring Boot та Next.js / Web application development for a library with Spring Boot and Next.js integration

## Архітектура та Технології

- **Backend:** Spring Boot (REST API, Spring Data JPA, Validation, Spring Security)
- **Frontend:** Next.js (App Router, SSR/SSG, API routes)
- **База даних:** MySQL
- **Тип взаємодії:** REST API
